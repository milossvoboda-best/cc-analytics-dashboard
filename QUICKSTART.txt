  • Interruptions: 5% šanca per segment (ak zapnuté)


═══════════════════════════════════════════════════════════════════════
  KĽÚČOVÉ METRIKY - VZORCE
═══════════════════════════════════════════════════════════════════════

  AES (Agent Effectiveness Score):
  --------------------------------
  AES = 0.25×Sentiment + 0.30×Compliance + 0.30×Resolution + 0.15×Quality

  ACI (Agent Consistency Index):
  ------------------------------
  ACI = 100 - (100 × CV)
  kde CV = StdDev(AES) / Mean(AES)

  FCR (First Contact Resolution):
  --------------------------------
  FCR = True keď resolution="full" AND callback=False AND escalated=False

  EPR (Escalation Prevention Rate):
  ----------------------------------
  EPR = (Prevented / Total) × 100%

  Compliance Risk:
  ----------------
  Risk = Critical_Fields×5 + Weighted_Fields + Violations×7

  TROUBLESHOOTING

  ❌ "Module not found" error
     → pip install -r requirements.txt

  ❌ Streamlit sa nespustí
     → pip install --upgrade streamlit

  ❌ Prázdny dashboard
     → Over filtre v sidebar (reset na "All")
     → Regenerate dataset v Config

  ❌ Timeline sa nezobrazuje
     → Vyber iný hovor (niektory môžu mať málo segmentov)

  ❌ Port 8501 už používaný
     → streamlit run app.py --server.port 8502

  DEMO SCENÁR (PRE DECISION MAKERS)
═══════════════════════════════════════════════════════════════════════

  1. ÚVOD (1 min)
     "Toto je CC Analytics dashboard s live dátami z 200 hovorov."

     - Compliance: Kde máme riziko? (ukáž violations)

     Prejdi na Agents tab
     - Ukáž top performers (ACI > 85)
     - Vyber agenta → jeho trend
     - "Vidíme konzistenciu vs. variabilitu"
  
  4. CALL TIMELINE (3 min)
     Prejdi na Calls tab
     - Vyber complaint call
     - Ukáž timeline:
       * Agent vs customer speaking
       * Silence periods (hold = problém)
       * Sentiment markers (Start negative → End positive)
     - "Real-time vizualizácia každého hovoru"
  
  5. FILTROVANIE & INSIGHTS (1 min)
     - Filtruj topic="technical" → ukáž dlhší AHT
     - Filtruj team="Support" → ukáž ich FCR
     - "Môžeme drill-down do ľubovoľného segmentu"
  
  6. ZÁVER (30 sec)
     "Všetko offline, syntetické dáta. Ready pre integráciu
